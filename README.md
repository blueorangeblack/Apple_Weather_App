# Apple Weather App

## 개발 목표
1. OpenWeather API를 사용하여 Apple Weather App과 비슷하게 만들기
2. CollectionView Compositional Layout사용하기
3. SwiftLint 이외에 다른 외부 라이브러리 사용하지 않기
<br>
